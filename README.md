# StudentProject

ASP.NET Core REST API для управления событиями.

## Сборка и запуск

1. Откройте терминал в корне проекта.
2. Соберите проект: dotnet build
3. Запустите проект: dotnet run
4. После запуска API будет доступен по адресу, указанному в консоли (например, https://localhost:5001/).

## Swagger

Для быстрого тестирования и знакомства с API используйте Swagger UI по адресу: https://localhost:PORT/swagger
*(PORT – фактический порт, выбранный при запуске; смотрите вывод запуска приложения)*

## Реализованные эндпоинты

### Получение всех событий GET /events
### Получение события по id GET /events/{id}
### Создание нового события POST /events
### Обновление события PUT /events/{id}
### Удаление события DELETE /events/{id}
